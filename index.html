Installations-Konfiguration

Diese JSON-Datei wird an den Installer übergeben.

📄 install_config.json

Die Konfiguration wird live aktualisiert. Lade sie herunter oder kopiere sie und übergib sie mit --config install_config.json an den Installer.

Installationsbefehl

Führe diesen Befehl auf deinem Server aus.

🚀 curl | sudo bash

$

Erfordert root-Zugriff. Der Befehl lädt den Bootstrap-Installer herunter und führt alle Systemprüfungen aus, bevor die Installation startet.
